Navegadores web

Un navegador web (en ingles, web browser) es un software, aplicacion o programa que permite el acceso a la Web, interpretando la informacion de distintos tipos de archivos y sitios web para que estos puedan ser vistos

La funcionalidad basica de un navegador web es permitir la visualizacion de texto, posiblemente con recursos multimedia incrustados

Ejemplos de navegadores web

Safari:Safari es un navegador web de codigo cerrado desarrolado por Apple Inc. Esta diselado para macOS,IpadOS e IOS

Mozilla Firefox es un navegador web libre y de codigo abierto desarrolado para Linux,Android,IOS,macOS y Microsoft Windows coordinado por la Corporacion Mozilla y la fundacion Mozilla

Google Chrome es un navegador web de codigo cerrado desarrollado por Google aunque derivado de proyectos de codigo abierto
